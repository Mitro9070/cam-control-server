# Project_R1

Проектная структура для реализации R1 по утвержденному ТР.

Главный принцип:
- строгая структура,
- логический порядок,
- изоляция частей по отдельным папкам.

Секции:
- `01_ASCOM_Local_Server`
- `02_Python_Camera_Service`
- `03_GreatEyes_SDK_Adapter`
- `04_Storage`
- `05_UI`
- `06_Testing`
- `07_Deployment_and_Ops`
- `08_Docs_and_Artifacts`
