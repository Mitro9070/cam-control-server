# Быстрый деплой на ПК камеры

## Цель
- Развернуть рабочий контур `ASCOM Local Server + Python camera-service` на целевом ПК.
- Минимизировать ручные шаги.

## Предусловия
- Windows, установленный ASCOM Platform 7.x.
- Build Tools 2022 (MSBuild).
- Python 3.11+.
- Репозиторий `Project_R1` скопирован на целевой ПК.

## Шаги
1. Открыть `PowerShell` от имени администратора.
2. Выполнить:
   - `cd <path>\Project_R1\07_Deployment_and_Ops`
   - `.\deploy_camera_pc.ps1`
3. Проверить `.env`:
   - `02_Python_Camera_Service\.env`
   - обязательно заполнить `SDK_DLL_PATH` для `SDK_MODE=native`.
4. Запустить сервис:
   - `.\start_camera_service.ps1`

## Проверка
- Health:
  - `http://127.0.0.1:3037/api/v1/health`
- UI:
  - `http://127.0.0.1:3037/ui/`
- ASCOM:
  - в ASCOM Chooser должен быть `GreatEyes 9.0 (Project_R1 ASCOM Camera)`.

## Примечания
- `RegAsm` warning `RA0000` допустим для локального стенда (неподписанная сборка).
- Для production распространения рекомендуется подписать .NET сборку и использовать инсталлятор.
