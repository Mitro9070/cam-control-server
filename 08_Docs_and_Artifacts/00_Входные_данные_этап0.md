# Входные данные этапа 0 (зафиксировано)

## Камера и SDK
- Целевая модель: `GreatEyes 9.0`.
- Исходные SDK-артефакты и примеры интеграции: в папке `repos`.
- Рабочее допущение: использовать существующие сигнатуры DLL из текущих примеров, уточнения будут вноситься по мере получения новых данных.

## Клиенты для проверки
- `NINA`
- `MaxIm DL`
- `SGP`
- Собственный клиент (в рамках `Project_R1`)

## Backend и БД
- Старт backend выполняется сразу с PostgreSQL.
- Конфигурация БД хранится через `.env` (локально), без публикации секретов в кодовой базе.

## S3
- Используется как внешнее хранилище артефактов проекта.
- Обязательное правило: внутри бакета создавать отдельную папку проекта и соблюдать структуру каталогов.
