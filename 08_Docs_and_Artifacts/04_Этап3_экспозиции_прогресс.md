# Этап 3: Экспозиции и получение кадра (прогресс)

## Реализовано
- Эндпоинты управления экспозицией:
  - `POST /api/v1/camera/exposures`
  - `GET /api/v1/camera/exposures/{id}/status`
  - `POST /api/v1/camera/exposures/{id}/abort`
  - `POST /api/v1/camera/exposures/{id}/stop`
- Эндпоинт кадра:
  - `GET /api/v1/camera/images/latest`
- Конфигурация ROI/binning:
  - `PUT /api/v1/camera/config/roi-binning`

## Поведение
- В mock-контуре экспозиция выполняется асинхронно в фоне.
- `image_ready` и `camera_state` обновляются через runtime state machine.
- После завершения формируются технические метаданные кадра + sample pixels.
