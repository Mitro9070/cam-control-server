# Технический долг (backlog)

## Зафиксированные вводные
- GreatEyes SDK: `Instruction Manual Version 12 Rev.5` (по данным команды).
- Использовать эту версию как текущий ориентир при валидации native-контура.

## Техдолг (отложенные задачи)
1. **Native SDK контур**
   - Провести полную валидацию на реальном железе для операций:
     - connect/disconnect
     - exposure flow
     - cooling/warm-up
   - Сверить поведение с SDK `Version 12 Rev.5`.

2. **API-контракты**
   - Добавить `percent` в статус экспозиции.
   - Зафиксировать финальный wire-format для `ImageArray` (Python -> C#).
   - Унифицировать endpoint warm-up status (финальный путь по ТР).

3. **База данных**
   - Расширить операционные таблицы выше baseline:
     - lifecycle экспозиций,
     - технические метаданные кадров,
     - связность с сессиями.

4. **Тесты**
   - Расширить integration-тесты:
     - `Python service + real postgres`
     - `Python service + native SDK`
   - Добавить стабильный прогон тестов миграций для режимов `local/postgres`.

5. **C# сборка и регистрация**
   - Подтвердить сборку `ASCOM.ProjectR1.Camera` в Visual Studio.
   - Проверить COM registration/unregistration и видимость драйвера в ASCOM Chooser.
