# Этап 2: рабочая структура документов

Эта папка используется как единая точка хранения артефактов этапа 2.

Структура:
- `01_План_работ` - календарный и технический план реализации.
- `02_Приемка_и_тестирование` - чеклисты, тест-планы, протоколы прогонов.
- `03_Отчеты_приемки` - финальные и промежуточные отчеты.
- `04_Доработки_и_изменения` - спецификации изменений, ADR, notes по релизам.
- `99_Архив` - неактуальные/сырьевые материалы, перенесенные из рабочих папок.
