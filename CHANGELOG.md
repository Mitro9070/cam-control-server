## [Unreleased] — актуальное состояние

### Python camera-service (`02_Python_Camera_Service`)

- **FITS:** экспорт с выбором `BITPIX` **16** или **32**; настройка `fits_export_bitpix` в `GET/PUT /settings`, дефолт из env `FITS_EXPORT_BITPIX` (по умолчанию 32). Ответ `POST .../export/fits` содержит `fits_bitpix`, `sensor_bit_depth`.
- **Настройки:** слияние сохранённых значений с `DEFAULT_SETTINGS` (в т.ч. для старых записей без новых ключей).
- **PostgreSQL:** при недоступности БД — короткие таймауты подключения, деградация без падения `health`/`connect` (см. `storage.py`, `db.py`).
- **Сервис:** редирект `GET /ui` → `/ui/`; статика UI с абсолютными путями `/ui/...`.

### Web UI (`05_UI/web`)

- Дашборд статуса (сервис, линк камеры, температура `ccd_temp_c`, цель, мощность, экспозиция), техно-стили.
- Исправление отображения температуры: используется поле API `ccd_temp_c`.
- Спиннеры и прогресс экспозиции; настройка **Экспорт FITS (BITPIX)** на вкладке камеры.
